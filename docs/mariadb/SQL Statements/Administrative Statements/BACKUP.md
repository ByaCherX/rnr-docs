## BACKUP STAGE
The BACKUP STAGE commands are a set of commands to make it possible to make an efficient external backup tool.


```sql
--Syntax--
BACKUP STAGE [START | FLUSH | BLOCK_DDL | BLOCK_COMMIT | END ]
```
